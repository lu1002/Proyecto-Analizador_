# Proyecto-Analizador_

Nombre del proyecto: Traductor de Lenguaje Natural a Expresiones Matemáticas con jerarquia

Interpretar operaciones aritméticas escritas en lenguaje natural, las traduce a una expresión matemática válida, evalua y muestra su resultado respetando la jerarquía de operaciones.
